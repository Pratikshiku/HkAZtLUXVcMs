### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户（学生）、团长、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 活动信息
5. 公告信息
6. 社团信息
7. 个人中心
8. 后台管理


**团长功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 活动信息管理
6. 活动审核管理
7. 公告信息管理
8. 社团信息管理
9. 社团审核管理



**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 基础数据管理（活动类型管理、公告信息类型管理、社团类型管理）
5. 活动信息管理
6. 活动审核管理
7. 公告信息管理
8. 社团信息管理
9. 社团审核管理
10. 团长管理
11. 学生管理
12. 轮播图信息
